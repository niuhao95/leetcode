## 存储方式
1. 数组
  - 顺序存储
  - 扩容缩容问题 O(n)
  - 插入删除 O(n)
2. 链表
  - 链式存储
  - 分配内存空间存储节点指针
    - 插入删除 O(1)
3. 图
  - 邻接表 -> 链表
  - 邻接矩阵 -> 二维数组
4. 树
  - 堆(完全二叉树) -> 数组
  - 链表
